Google translate API v2

You can translate text from one language to another language by sending an HTTP GET request to its URI.

The URI for a request is 

https://www.googleapis.com/language/translate/v2

parameters

ParameterMeaningNotes
callback A JavaScript function to run when the response is received
  • This optional parameter allows you to specify a JavaScript function to handle query results for pure client-side implementations.
  • Embed the API query in <script> tags.
  • Define the callback function in <script> tags.
  • See REST from JavaScript in the Getting Started document for an example.
format Format
  • This optional parameter allows you to indicate that the text to be translated is either plain-text or HTML. A value of html indicates HTML and a value of text indicates plain-text.
  • Default: format=html.
key Your API key
  • This API requires a valid key for all requests
  • Get a key from the Google APIs console.
prettyprint Returns a response with indentations and line breaks
  • If prettyprint=true, the results returned by the server will be human readable (pretty printed).
  • Default: prettyprint=true.
q Source text
  • The text to be translated.
  • You can repeat this parameter more than once in a single request to translate multiple texts.
source Source language
  • The language of the source text. The value should be one of the language codes listed in Language reference.
  • Supported language codes is produced by languages
  • If a language is not specified, the system will attempt to identify the source language automatically.
target Target language
  • The language to translate the source text into. The value should be one of the language codes listed in Language reference
  • Supported language codes is produced by languages
translatedText Translated text
  • Read only Parameter
detectedSourceLanguage Detected Source Language code (if source is not specified explicity)
  • Read only Parameter
  • Optional
