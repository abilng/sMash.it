User

URL:http://example.com/api/user

It is a user

It use HTTP GET request with:
Inputs:
  • username
  • password
  • country
Produces:
  • token
Attribute Descriptions
  1. username: required, must be unique.
  2. password: required.
  3. country: country code(string)
    Producers:
    • cn
    • getcountry
  4. token: userID returned as result
Error Number and Description
  1. 550 :- no such user
  2. 504 :- Authorization failed
